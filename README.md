## ⚙️ Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/bitvora.git
```

### 2️⃣ Navigate to the Folder

```bash
cd bitvora
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run the Development Server

```bash
npm run dev
```

## 🏗 Build for Production
